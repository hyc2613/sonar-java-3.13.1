特定关键词：do/while/if/else/for/switch后面必须带空格

Noncompliant Code Example

    int i = 0;
    for(i = 0; i < 10; i++) { // Noncompliant
        // do something
    }

    if(i < 10) {  // Noncompliant
        // do something
    }

Compliant Solution

    int i = 0;
    for (i = 0; i < 10; i++) {
        // do something
    }

    if (i < 10) {
        // do something
    }
